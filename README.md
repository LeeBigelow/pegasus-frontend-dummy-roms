# Dummy roms for all systems to test [Pegasus-Frontend](https://pegasus-frontend.org/) themes

Sourced from: [ES-DE dummy files](https://gitlab.com/es-de/emulationstation-de/-/tree/master/tools/system-dirs-dummy-files)

Contains:
* Dummy rom files for most (all?) systems.
* Single simple all-in-one `metadata.pegasus.txt` for easy Pegasus testing
* The bash script `generate-pegasus-metadata.sh` that was used to convert all the ES-DE systeminfo.txt into a single `metadata.pegasus.txt` file. In case you wish to tinker.
