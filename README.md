# Dummy rom metadata files for testing [Pegasus-Frontend](https://pegasus-frontend.org/) themes

* `metadata.pegasus.txt` files provided for: all systems, only those with consolegame images, or only those with controller images. (Based on [Slatest](https://github.com/LeeBigelow/pegasus-theme-slatest) theme image set)
* Simple single all-in-one `metadata.pegasus.txt` files for easy testing
* Based on: [ES-DE dummy files](https://gitlab.com/es-de/emulationstation-de/-/tree/master/tools/system-dirs-dummy-files)
